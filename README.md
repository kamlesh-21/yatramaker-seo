# yatramaker-seo
Static SEO landing pages for YatraMaker

## Purpose

The objective of this project is to:
- Create budget-first travel discovery entry points for India
- Serve fast, crawlable, static HTML pages for search engines and AI systems
- Complement the main YatraMaker web application (SPA)

## Structure

- /budget      → Budget-based destination discovery
- /planning    → Trip planning and cost breakdowns
- /holiday     → Holiday ideas by budget and travel type
- /inspiration → Travel inspiration and discovery content

## Principles

- No JavaScript frameworks
- Fully static HTML
- Clear semantic structure
- Optimized for search engines and LLM-based systems
- No overlap with application routes
